1)cd путь - смена директории	
	cd .. - перейти в предыдущий каталог
	cd название папки - перейти в папку
	cd /d путь - переключение между дисками
2)dir путь - вывод списка файлов в директории	
    /b выводит только имена файлов
	/w вывод списка в несколько столбцов
	/p пауза после запонения каждого экрана
	/s отображает файлы из текущего каталога и всех его подкаталогов
3)tree путь - выводит директорию в древовидном виде(только папки)
	/f для вывода имен файлов
4)md название1 .. названиеN - создаст N папок с названиями, которые мы указали
	md 1\2\3\4\5 - создаст папку с названием 1, потом в папке 1 папку 2, в папке 2 папку 3...
5)rd  путь - удалит каталог(но только пустой)
	/s удаление дерева каталогов
	/q отключение подтверждения для удаления каталогов
6)del путь - удаляет файлы
	del путь\*.txt удалит все txt файлы
	del путь\*.* удалит все файлы
	/s удалит все указаные файлы из всех подкаталогов
7)copy путь1\имя файла путь2 - скопирует файл из путь1 в путь2(ТОЛЬКО ФАЙЛЫ)
	copy copy путь1\имя1 файла путь2\имя2 - при перемещениее изменит имя файла(также можно менять путь)
	copy путь\*.* путь2 - скопирует все файлы
	copy путь\a*.* - скопирует все файлы начинающиеся на букву a
8)xcopy путь1 путь2 - копирует файлы
	/e копирует подкаталоги
	/y копировать с заменой(кликает YES)
	xcopy путь1 путь2\названте новой папки\ - создаст папку и скопирует туда файлы
	/s копирует только не пустые каталоги
	/h копирует скрытые и системные файлы
	/o копирует сведенья о владельце объекта
9)robocopy 
10)move путь1 путь2 - перемещает файл из 1 в 2
	/y - переместить с заменой 
	также как и в copy можно переименовать
11)type путь - выводит текст из файла в cmd
12)find "что найти" путь\название файла - ищет строки в файле 
	/i не учитывать регистр
	/v вывести все, кроме подстроки, которую мы написали
13)For параметр(/L) %переменная in (набор) do (команда)
	Для чисел For /L %i in (1(начало), 1(шаг), 31(конец)) do (что нужно сделать)
	Для файлов For %i in (ПУть\*txt) do (действие) - перебор всех файлов txt
	Рекурсивный перебор For /r путь %i in (конструкция файла(*.txt)) do (действие)
	Для каталого For /D %i in (путь\*) do (действие)
14)ren имя1.тип имя2.тип - переименовать файл
15)FC файл1 файл2 - сравнить два файла.
